About the Project
Rex is an AI-powered real-time detection system designed to identify Distributed Denial-of-Service (DDoS) attacks with speed, precision, and adaptability. Built for modern networks, Rex monitors traffic patterns, learns from anomalies, and reacts instantly to potential threats. Whether used in research, corporate environments, or security education, Rex offers a flexible foundation for defending digital infrastructures.

Key Features
Real-time traffic analysis and threat detection
Machine learning models tailored for DDoS classification
Modular and lightweight architecture
CLI interface with optional dashboard integration
Easily extensible for custom use cases

Use Cases
Network security in corporate or academic settings
Cybersecurity research and experimentation
Educational projects and demonstrations
Prototype development for larger-scale defense systems

Objectives
Rex aims to simplify the complexity of intrusion detection systems by offering a smart, developer-friendly platform that merges data science with practical network defense. Its design focuses on clarity, efficiency, and performance without compromising detection accuracy.
